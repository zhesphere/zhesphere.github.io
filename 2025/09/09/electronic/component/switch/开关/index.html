轻触开关(Tactile Switch)

定义与核心特征

瞬时接触型电子开关

  • [瞬时动作]{.rainbow}：仅在施加外部压力(通常为手指按压)时导通电路，撤去压力后凭借内部弹性结构(多为金属弹片)自动复位断开电路。不具备自锁功能
  • [触感反馈]{.rainbow}：内部弹片结构在按压行程中产生形变，当达到临界点(操作力点)时突然“塌陷”，提供清晰可辨的“咔嗒”触感(Tactile Feedback)和声音反馈(Click Sound)，明确指示开关已被有效触发
  • [小行程、低操作力]{.rainbow}：按键行程(Travel)通常短小(0.1mm1mm)，所需操作力(Operating Force)较低(几十克力^1几百克力)，适合频繁、快速操作
  • [紧凑尺寸]{.rainbow}：体积小巧、封装形式多样(SMT、THT)，高度集成化，适应现代电子产品微型化需求

工作原理

  1. [初始状态]{.rainbow}(释放状态-off)
    • 内部拱形金属弹片(Dome)处于自然拱起状态
    • 弹片与底部固定触点(Common Terminal,C)保持接触，但与另一个固定触点(Normally Open Terminal,NO)之间存在微小间隙
    • 电路处于开路状态(C与NO断开)
  2. [按压过程]{.rainbow}
    1. 手指施加垂直向下的力于按键帽(Button/Plunger)
    2. 压力通过按键帽传递至弹片顶部
    3. 弹片在压力下逐渐向下弹性形变(储存势能)
  3. [临界触发点]{.rainbow}(操作力点-Actuation Point)
    1. 当压力达到并超过弹片的临界屈曲力时，弹片结构失稳，瞬间从拱形“塌陷”成反向凹形或平直状态
    2. 此瞬间动作产生明显的“咔嗒”触感和声音
    3. 弹片底部(或与弹片相连的动触点)迅速与NO触点接触
    4. 电路导通(C与NO连通)
  4. [复位过程]{.rainbow}(释放状态恢复-Reset)
    1. 撤去手指压力
    2. 储存于弹片中的弹性势能释放。驱动弹片迅速向上回弹
    3. 弹片恢复初始拱形状态，脱离与NO触点的接触、
    4. 电路恢复开路状态(C与NO断开)
      关键点：弹片的材料、形状(圆形、十字形、四腿形等)、厚度、拱高设计是决定操作力、行程、触感、回弹力、寿命的核心因素

结构

典型SMT贴片轻触开关：

  1. [上盖/外壳]{.rainbow}(Housing/Cover)
    • 材质：耐高温绝缘材料(如PBT、LCP、PA66)，提供结构支撑、电气绝缘和环境保护
    • 功能：固定内部组件，提供按键帽的导向和行程限制，防止灰尘、液体侵入(防护等级依据设计而定)
  2. [按键帽/柱塞]{.rainbow}(Button/Plunger)
    • 材质：工程塑料或金属(不锈钢)，顶部常覆有硅胶或其他材质材料以提升触感或密封性
    • 功能：直接承受用户按压，将力传递至弹片，其形状(平顶、凸顶、十字形)影响按压体验
  3. [金属弹片]{.rainbow}(Dome/Snap Disc)
    • 材质：不锈钢(SUS 301、SUS 304最常见)，有时镀金(Au)或镀银(Ag)以提升导电性和耐腐蚀性
    • 功能：提供触感反馈、触发电路通断、产生回弹力
  4. [基板/底座]{.rainbow}(Base)
    • 材质：绝缘材料(如FR4环氧树脂板、耐高温塑料)或金属(接地用)
    • 功能：承载固定触点(端子)，提供与PCB焊接的界面
  5. [固定触点]{.rainbow}(Terminals)
    • 端子类型：SMT贴片端子(鸥翼型、J型)、DIP插件端子。通常为黄铜或磷青铜基材，表面镀层(锡、金、银)以提高可焊性、导电性和耐腐蚀性
    • 触点定义：
      • 公共端(Common,C/COM)：始终与弹片(动触点)接触
      • 常开端(Normally Open,NO)：常态下与弹片(动触点)断开，按压导通时连通
      • (部分开关有常闭端(Normally Closed,NC)，轻触开关中较少见)
  6. [内部空间/腔体]{.rainbow}(Cavity)
    • 功能：容纳弹片并提供其动作空间。设计影响弹片动作顺畅度和防尘能力
  7. [辅助部件]{.rainbow}(可选)
    • 密封圈(Sealing Ring/Gasket)：硅胶或橡胶材质，用于提升防水防尘等级(如IP67)
    • 粘性背胶(Adhesive Tape)：用于面板安装固定
    • 防护盖/薄膜(Protective Cover/Film)：防止异物进入或作为面板的一部分
      结构设计趋势
      更薄型化(Low Profile)、更小尺寸、更高防护等级、更优触感一致性、更易自动化贴装

类型

  1. [按安装方式]{.rainbow}
    1. 贴片型(SMT-Surface Mount Technology)
      • 特点：端子为金属焊盘，通过回流焊工艺贴装在PCB表面。主流封装形式
      • 优势：节省PCB空间(无通孔)，自动化贴装效率高，适合高密度组装
      • 常见尺寸：3x3mm, 3x4mm, 4x4mm, 4.5x4.5mm, 5x5mm, 6x6mm等
    2. 插件型(THT-Through-Hole Technology)
      • 特点：端子为金属引脚，插入PCB通孔后焊接
      • 优势：机械强度更高，焊接点更牢固，维修更换相对容易
      • 常见封装：DIP(Dual In-line Package)
  2. [按按键高度/行程]{.rainbow}
    • 超薄型(Ultra Low Profile)：高度<1.0mm(如0.65mm)，行程极小。用于超薄设备(手机、平板)
    • 薄型(Low Profile)：高度1.0mm~2.5mm。主流高度，平衡了薄型化和良好触感
    • 标准型(Standard Profile)：高度>2.5mm。行程和操作力可能更大，触感更明显
  3. [按按键顶部形状]{.rainbow}
    • 平顶型(Flat)：最常见，与面板齐平或微凸
    • 凸顶型(Domed)：顶部圆弧凸起，便于手指定位
    • 十字型/开孔型(Cross/Holed)：中心有孔或十字槽，便于安装定位销或透光(用于背光开关)
  4. [按触感反馈特性]{.rainbow}
    • 标准触感型：具有清晰“咔嗒”感
    • 软触感型(Soft Tactile)：“咔嗒”感更柔和、安静
    • 无声型/静音型(Silent)：通过特殊弹片设计(如硅胶圆顶Silicon Dome)或阻尼结构，极大减弱或消除点击声，仅有触感变化
  5. [按防护等级]{.rainbow}
    • 标准型：无特殊密封，防水防尘能力弱
    • 防尘型：带基本防尘结构
    • 防水型：带有密封圈(O-ring)
  6. [按特殊功能]{.rainbow}
    • 带灯型(Illuminated)：集成LED(通常为SMD LED)，提供背光指示功能，需额外供电引脚
    • 带接地端子型：增加接地脚(Ground Pin)，用于ESD防护或屏蔽
    • 多向按键型(Multi-directional)：单一开关支持多个方向按压(如上、下、左、右、中)，内部结构更复杂(如五向摇杆开关)

关键材料

材料选择直接影响开关的性能、寿命、可靠性和成本

  1. [外壳/上盖材料]{.rainbow}
    1. PBT(聚对苯二甲酸丁二醇酯)：常用，耐热性好(>200℃)，尺寸稳定，耐化学性好，成本适中
    2. LCP(液晶聚合物)：超耐高温(>260摄氏度)，极低吸湿性，尺寸稳定性极佳，流动性好(利于薄壁成型)，成本高。用于高端、小型化、耐高温回流焊要求严苛的产品
    3. PA6/PA46(尼龙)：韧性好，耐磨损，成本较低，但吸湿性较大，尺寸稳定性略逊于PBT/LCP
    4. 金属(不锈钢)：用于需要极高强度、EMI屏蔽或特殊外观的场合
  2. [弹片材料]{.rainbow}
    1. 不锈钢(SUS 301,SUS 304)：绝对主流材料
      • SUS 301：硬度高(HV 430-550)，弹性好，回弹力强，寿命长。是要求高操作力、清晰触感和长寿命开关的首选
      • SUS 304：硬度较低(HV 350-450)，耐腐蚀性更优，成本略低。常用于对操作力要求稍低或更强调耐腐蚀的环境
    2. 表面处理：为提高导电性、耐腐蚀性和耐磨性，常进行电镀
      • 镀金(Au)：接触电阻最低且最稳定，耐腐蚀性极佳，成本最高。用于高可靠、低电平信号(电信、医疗、精密仪器)
      • 镀银(Ag)：导电性好，成本低于镀金，但易硫化发黑(特别是在含硫环境中)
      • 镀锡(Sn)：成本最低，可焊性好，但耐磨性和耐腐蚀性一般。常用于低端产品或不直接作为接触面的基材
  3. [基板/底座材料]{.rainbow}
    • FR4环氧玻璃纤维板：最常用，绝缘性好，强度高，耐热性满足一般回流焊要求
    • 高温塑料(如PBT,LCP)：用于一体化注塑成型的底座
    • 金属基板(如铝)：用于需要散热或接地屏蔽的场合
  4. [端子材料]{.rainbow}
    1. 基材：黄铜(Brass,H62/H65)或磷青铜(Phosphor Bronze,C5191/C5210)
      • 黄铜：导电性好，成本低，强度尚可
      • 磷青铜：弹性、抗疲劳性、耐腐蚀性优于黄铜，导电性略低，成本稍高。常用于需要良好插拔耐久性或弹性接触的端子
    2. 端子表面镀层：与弹片镀层类似，常用锡(Sn)、金(Au)、银(Ag)。选择需考虑可焊性、耐腐蚀性、成本和接触电阻要求
  5. [密封/辅助材料]{.rainbow}
    • 硅橡胶(Silicone Rubber)：耐高低温(-60℃~200℃)，弹性好，化学惰性，用于防水密封圈(O-ring,Gasket)或软质按键帽
    • 聚氨酯(Polyurethane)：耐磨性、耐油性好，有时用于特殊密封件
    • 粘合剂：耐高温双面胶带(如3M VHB)用于面板粘贴安装

参数(==选型==)

  1. [电气参数]{.rainbow}(Electrical Characteristics)
    • 额定电压(Rated Voltage)：开关能安全可靠工作的最大电压(DC或AC)。常见12VDC,24VDC,48VDC
    • 额定电流(Rated Current)：开关触点能承受的最大连续电流。范围从毫安级(信号开关)到几百毫安(如50mA,100mA,200mA)。高电流会显著降低寿命并增加发热风险
    • 接触电阻(Contact Resistance)：触点闭合时两接触点间的电阻。关键参数！要求尽可能低(通常<100mΩ，优质品<50mΩ)且稳定。受触点材料、镀层、接触压力和污染影响
    • 绝缘电阻(Insulation Resistance)：开关断开触点间或触点与外壳间的电阻(通常>100MΩ@100VDC)。反映绝缘性能
    • 耐电压/介电强度(Dielectric Strength/Withstand Voltage)：开关断开触点间或触点与外壳间能承受而不击穿的最高电压(如250VAC for 1min)。反映绝缘耐压能力
  2. [机械参数]{.rainbow}(Mechanical Characteristics)：
    • 操作力(Operating Force)：使开关触发(弹片塌陷)所需的最小力。单位：gf(克力)或N(牛顿)。直接影响按压手感(轻触或需用力)。范围广泛(160gf,180gf,250gf,350gf等)
    • 回弹力(Return Force)：按键释放后，弹片恢复原状产生的力。应足够保证可靠复位
    • 总行程(Total Travel)：按键从自由状态到完全按到底的移动距离
    • 预行程(Pre-travel)：从按键开始下压到达操作力点(触发点)的移动距离
    • 过行程(Overtravel)：从操作力点到按键被完全按到底(被限位)的移动距离。提供一定的操作余量
    • 操作力点(Actuation Point/Tactile Point)：弹片塌陷、电路导通的精确位置(通常以行程百分比表示，如50%Total Travel)
    • 释放点(Release Point)：按键回升过程中，弹片开始复位、电路断开的精确位置。通常略低于操作力点(存在差值Hysteresis)
    • 触感(Tactility)：操作力-行程曲线中出现的明显“突降”或“峰值”，表征触感强度(Peak Force-Operating Force)。量化或主观描述(清晰、柔和、无感)
  3. [耐久性与可靠性]{.rainbow}(Durability & Reliability)
    • 机械寿命(Mechanical Life)：无负载(空载)条件下，开关能承受的操作次数(通常数十万至数百万次)。反映纯机械结构的磨损寿命
    • 电气寿命(Electrical Life)：在额定负载(电压、电流)条件下，开关能可靠完成通断操作的次数(通常数万至数十万次)。远低于机械寿命，受触点电弧烧蚀、材料转移、污染影响极大
    • 工作温度范围(Operating Temperature Range)：开关能正常工作的环境温度范围(如-40℃~+85℃)。材料(尤其塑料和弹片)需在此范围内保持性能稳定
    • 防护等级(Ingress Protection Rating-IPXX)：如IP67(完全防尘，可短时浸水)。由密封结构决定
    • 耐焊接热(Solder Heat Resistance)：SMT开关承受回流焊温度曲线的能力(如260℃ for 10s)。关键材料(塑料、弹片)需耐受此高温而不变形退化
    • ESD保护(ESD Immunity)：抵抗静电放电破坏的能力(通常按IEC 6100-4-2标准测试，如±8kV接触放电)。设计(如接地路径)和材料(绝缘性)影响ESD表现
  4. [环境耐受性]{.rainbow}(Environmental Resistance)
    • 耐高温高湿/冷热冲击：开关在严苛温度变化或循环下的性能稳定性
    • 耐化学品：抵抗清洁剂、溶剂、汗液等侵蚀的能力
    • 耐振动/冲击：在运输或使用过程中抵抗机械应力的能力

选型指南

  1. 电气需求
    • 负载特性：电流/电压大小？(最重要！信号级选小电流开关，功率级慎选并留余量)直流(DC)还是交流(AC)？负载性质(阻性、容性、感性-感性负载最易产生电弧)？
    • 信号完整性：是否需要极低且稳定的接触电阻？(高精度测量、微弱信号)——选镀金触点
    • 电路设计：需要几组触点(常开NO)？端子形式(SMT/THT)？引脚排布？
  2. 机械与操作需求
    • 操作感受：期望的操作力大小？行程长短？触感类型(清晰、柔和、静音)？声音要求(有声/静音)？
    • 尺寸限制：PCB允许的安装面积(长×宽)？整机允许的总高度？按键帽尺寸和形状？
    • 安装方式：SMT贴片 or THT插件？面板粘贴安装(需带背胶)？
  3. 环境与可靠性需求
    • 工作环境：湿温度范围？有无粉尘、液体(水、油)？是否接触化学品、汗液？有无强烈振动、冲击？——决定防护等级(IPXX)、材料选择(耐化性)、结构强度
    • ESD风险：是否在易产生静电的环境？是否需要特殊ESD防护设计(如接地脚)？
    • 寿命要求：预期操作次数？机械寿命要求？电气寿命要求(在何种负载下)？——选择寿命规格远高于实际需求的型号，尤其在负载较重时
    • 法规认证：是否需满足特定行业标准(如汽车AEC-Q200，医疗ISO 13485)？
  4. 成本与供应链
    • 预算限制
    • 采购批量：标准品 or 定制？
    • 供应商资质：行业口碑、质量体系认证(ISO 9001)、技术支持能力、交货期稳定性

流程建议

  1. 明确负载电流/电压(最优先！)
  2. 确定尺寸(长宽高)和安装方式(SMT/THT)限制
  3. 定义操作力和触感偏好
  4. 评估环境条件(温湿度、防护等级)
  5. 设定寿命和可靠性目标
  6. 筛选供应商提供的符合上述核心要求的型号清单
  7. 对比关键参数(接触电阻、寿命数据、防护等级)、成本、样品实测手感
  8. 考虑进行小批量环境/寿命验证测试

常见误区

  • 只关注尺寸和价格，忽略负载电流匹配——导致触点迅速烧蚀失效
  • 混淆机械寿命和电子寿命——对带负载的实际寿命期望过高
  • 忽视环境因素(如湿气、腐蚀)对接触电阻和寿命的影响
  • 未实际按压样品，仅凭参数选型——触感体验不符合预期
